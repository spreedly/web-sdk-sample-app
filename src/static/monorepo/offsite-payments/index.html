About Offsite Payments

Offsite payments require redirecting customers to a third-party site (like PayPal) to complete payment. Spreedly handles the redirect flow and returns the customer to your site with the payment method token.

Flow Steps
  1. Load SDK and initialize with auth params
  2. Select offsite payment gateway
  3. Call setupOffsitePayment() with config
  4. Call submitOffsitePayment()
  5. SDK creates payment method and redirects
  6. Use token for purchase API call
