About Stripe APM

Stripe Alternative Payment Methods (APMs) allow customers to pay using local payment methods like iDEAL (Netherlands), Bancontact (Belgium), SEPA Direct Debit, and more.

Flow Steps
  1. Create pending purchase on Spreedly
  2. Get client_secret from response
  3. Initialize SpreedlyStripeAPM
  4. Mount Stripe Payment Element
  5. User selects payment method
  6. Confirm payment - redirect to provider
  7. Complete payment and return
