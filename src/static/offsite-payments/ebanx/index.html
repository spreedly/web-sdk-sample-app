About EBANX

EBANX enables Latin American local payment methods for global merchants. Supports Oxxo (Mexico), Boleto Bancário (Brazil), Pix, and more.

Flow Steps
  1. Load SDK and fetch auth params
  2. Select payment method (e.g., Oxxo)
  3. Call setupOffsitePayment()
  4. Call submitOffsitePayment()
  5. Receive payment method token
  6. Create purchase transaction
  7. Redirect to payment provider
  8. Complete payment and return
